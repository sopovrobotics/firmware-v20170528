## Hardware

1. Motor-reducer Gekko MR12-100 (2 pieces)
2. Track Pololu 30T
3. Pololu 5v Step-Up/Step-Down Voltage Regulator S7V7F5
4. DRV8833 Dual Motor Driver Carrier
5. OprangePi Zero
6. Case for accumulator 18650
7. Accumulator 18650
8. Switcher
